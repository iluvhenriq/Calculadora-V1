# Calculadora V1

Calculadora com interface gráfica feita em Python com CustomTkinter.

## Funcionalidades
- Soma
- Subtração
- Multiplicação
- Divisão
- Tratamento de erro para divisão por zero

## Tecnologias
- Python 3
- CustomTkinter

## Como rodar
1. Instala as dependências:
```
pip install customtkinter
```
2. Roda o programa:
```
python main.py
```
