# Calculadora V1

Calculadora com interface gráfica feita em Python com CustomTkinter.

## Funcionalidades
- Soma
- Subtração
- Multiplicação
- Divisão
- Tratamento de erro para divisão por zero

## Como rodar
### Opção 1 — Executável
Baixa o arquivo `main.exe` na pasta `dist` e clica duas vezes.

### Opção 2 — Código fonte
1. Instala as dependências:
```
pip install customtkinter
```
2. Roda o programa:
```
python main.py
```

## Tecnologias
- Python 3
- CustomTkinter
